- EP-03 家庭服务记录：docs/backlog/epics/epic-03-services.md
- EP-04 活动与报名：docs/backlog/epics/epic-04-activities.md

- EP-06 权限与安全：docs/backlog/epics/epic-06-permissions-security.md
